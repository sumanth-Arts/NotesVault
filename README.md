# NotesVault – College Notes Portal

NotesVault is a Flask-based web application built to make studying simpler, faster, and more effective for students.

Many students start preparing for exams at the last moment and often struggle to find proper, concise notes. This platform solves that problem by providing short, clear, and exam-focused notes that cover the important concepts required to understand subjects and pass exams confidently.

## Features

* Semester-wise subject organization
* Unit-wise structured notes
* Topic-wise explanations with definitions and examples
* Problem Bank with solved questions
* PPT / PDF / Question Bank viewer
* Admin Panel for managing notes and resources
* Announcements section
* Responsive design for mobile and desktop

## Tech Stack

* Python
* Flask
* SQLite
* HTML
* Tailwind CSS
* JavaScript
* Render (Deployment)
